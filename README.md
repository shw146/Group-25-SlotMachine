# Group-25-SlotMachine
This is the repository for KnowIdea's work in creating and analyzing separate generations of the same prompt given to a generative AI system.

We are going to be using Anthropic's Claude Code. 

As of 4/12/26 the latest version is version 2.1.105 which we will be using as the harness for our work.

## Below is a list of steps that you should take when starting on your assigned generations:
1. Read through this whole list before you start working! Trust me, there are some weird specifics that we have to look out for.
2. Read through the entirety of the canvas assignment. It is a lot of information, and much of it is scattered throughout the page, so it is important to read through everything and make sure you understand what is going on!
3. We will be using claude code as mentioned above. Thus, please make sure you have made a claude account, but do not start working on it until you are fully ready to work! Claude has recently enforced a 5 hours time limit during certain hours, so if you start and do not finish you may run out of time and we don't want that.
4. You have been assigned a number of candidates to generate. Please make directories in this repo in the appropriate step (1,2,3,etc.) to store your work in.
5. Once you are fully ready and have reviewed all of the things that you will need to record down as you are working (as described on canvas), feel free to work as you'd like!

If you run into any questions, please feel free to shoot a message on slack!
