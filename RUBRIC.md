# Rubric

## This is the rubric for the AI generated slot machines we are going to score:

|Category|Description|Examples|Points|
|---|---|---|---|
|Functionality|Does the program actually run properly?|Wheel spins, credit updates, proper input/output, etc.|25|
|Code quality|Is the code clean and concise while containing all the necessary information?|HTML/CSS separation, modular coding, consistent stylizing, comments for documentation, etc.|20|
|Thematic consistency|Does the program make sense as a slot machine?|risk vs. reward, do outputs make sense, is there an indication of how to use the machine?|25|
|Accessbility|Is this program accessible to those who need it?|High contrast, screen reader compatibility, sounds, etc.|5|
|UI/UX|Does the program look and feel good to use?|Button responsiveness, engaging rather than boring, etc.|25|

When scoring each generation, please use these categories to give a score out of 100. 

**If there is something that seems interesting and is not included in this rubric, please bring it up on slack!**
